Harvest Pierce County Gleaning Site

Redesigning for community impact, accessibility, and operational sustainability.

Project Overview

This project involved a full redesign of the Harvest Pierce County Gleaning Program’s website to increase volunteer engagement, modernize workflows, and enhance accessibility for a broad community user base. Our redesign process involved wireframing, form optimization, copywriting, and collaboration with staff and volunteers.

Role

UX Designer & Developer

  • Created a new IA and navigation system to improve clarity and reduce user friction
  • Designed and coded HTML forms for volunteer signups with accessible error handling
  • Incorporated feedback from multiple user groups, including staff and first-time volunteers
  • Maintained legacy data paths while introducing responsive, mobile-first layouts

Technology & Tools

  • HTML/CSS/JS
  • Formspree (form handling)
  • Google Fonts & SVG graphics
  • Figma (wireframes and layout concepts)
  • GitHub & Netlify (for version control and deployment)
