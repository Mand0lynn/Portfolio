USAF SharePoint Permissions

Designing secure, automated access workflows for sensitive HR environments.

Project Overview

This project explores how secure, scalable workflows can be implemented within SharePoint to support HR operations subject to HIPAA and CUI regulations. I worked with my teammate to developed a demo environment using Power Automate and Microsoft 365 to model role-based access control across departments. The goal was to create a functional and compliance-aligned solution for managing sensitive personnel data. The work is part of my internship with the U.S. Air Force.
